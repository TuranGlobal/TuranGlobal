# 🌍 TURAN

## About Turan

**Turan** is a decentralized financial infrastructure that combines digital asset intake, liquidity management, and a reward system within a unified protocol.

The platform allows users to participate using ETH, USDC, or TRN. All incoming funds are automatically distributed according to predefined directions: liquidity formation, investment allocation, ecosystem development, and operational support.

The core element of the system is the **TRN token** — a utility asset used for participation activation, investment accounting, and reward distribution.

Turan is built on principles of **transparency, automation, and minimization of intermediaries**. To understand Turan more deeply, it is important to look beyond its functions and focus on the economic model it creates.
